## 🔭 Featured Projects

### PayCollect — Fintech Payment Collection Platform
> Production SaaS · React + Node.js + MongoDB · Live at [get-pay-collect.com](https://get-pay-collect.com)

Built a full-stack merchant payment collection platform from scratch for a fintech client. Enables businesses to send payment requests, track collections, and reconcile payments in real time.

**What I built:**
- Complete merchant onboarding flow — Google OAuth signup → card capture → Stripe Connect → dashboard
- 4-layer Stripe payment architecture: payment link generation, app-open sync, daily cron reconciliation, webhook auto-update
- SMS notifications via AWS SNS (Mumbai region) — fires on payment request creation and reminders
- Transactional email via Resend with custom domain (`payments@get-pay-collect.com`)
- Bulk Excel/CSV upload for customers and payment requests with downloadable templates
- Custom domain, SSL, and Cloudflare email routing setup
- EN/ES bilingual US-focused landing page

**Stack:** React · Vite · Node.js · Express · MongoDB · Stripe · AWS SNS · Resend · Vercel · Render
